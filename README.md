
# Labewbew
Trang web này là sân chơi bổ ích dành cho những người có niềm đam mê mãnh liệt với Roblox. Trang web của nhóm Labewbew chúng em hướng đến việc tạo nên một cộng đồng Roblox VN thân thiện - sáng tạo - kết nối, nơi người chơi có thể khám phá game, nạp Robux và giao lưu cùng nhau trong một giao diện đẹp mắt, đơn giản và dễ sử dụng.

## Cách chạy
- Tải dự án về máy hoặc clone repo.
- Mở file index.html bằng trình duyệt.
- Nếu dùng VS Code, có thể mở bằng Live Server để xem nhanh và tự động cập nhật.
